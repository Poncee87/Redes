¿Qué es el cableado horizontal?

El cableado horizontal consiste de dos elementos básicos: Cable Horizontal y Hardware de Conexión (también llamado "cableado horizontal") que proporcionan los medios básicos para transportar señales de telecomunicaciones entre el área de trabajo y el cuarto de telecomunicaciones.

¿Qué es el cableado vertical o backbone?

El cableado backbone, cableado vertical o troncal es una parte del cableado estructurado de un edificio, sirviendo de conexión entre el rack de telecomunicaciones principal (MDF o Main Distribution Frame) y el rack de telecomunicaciones secundario (HCn u Horizontal Cable) que se distribuyen por el edificio.

¿tipos de cables para el sistema de cableado horizontal?

UTP (par trenzado sin apantallar): Es el más común en redes locales. STP (par trenzado apantallado): Ofrece mayor protección contra interferencias. Fibra óptica: Se usa para mayor ancho de banda y menores distancias.

¿Normativa para hacer cables de red clase A y B?

Para hacer cables de red de Clase A y B, debes seguir las normas internacionales como ISO/IEC 11801 y ANSI/TIA-568, que definen los requisitos para el cableado de telecomunicaciones y redes de área local. La ISO/IEC 11801 cubre especificaciones para cables de categorías 5e y 6, mientras que la ANSI/TIA-568 detalla estándares similares y su instalación. También es importante considerar la norma IEC 61156 para cables de datos y la ISO/IEC 60603-7 para conectores modulares. Clase A generalmente se refiere a cables de categoría 5e, y Clase B a cables de categoría 6.
