¿Que son
Los componentes pasivos en la red?

Un componente pasivo, o dispositivo pasivo, es un componente electrónico que solo puede recibir energía, que puede disipar, absorber o almacenar en un campo eléctrico o magnético. Los componentes pasivos no necesitan ningún tipo de energía eléctrica independiente para funcionar.

Componentes Pasivos:

RJ45

Conector estándar usado para cables de red Ethernet.



Fibra óptica

Medio de transmisión que usa filamentos de vidrio o plástico para transferir datos a alta velocidad.

Keystone

Conectores modulares instalados en placas o patch panels, permitiendo la conexión de cables de red.

Cable UTP

Cable de par trenzado sin blindaje utilizado para conexiones de red.

Patch Panel

Panel que organiza y facilita la conexión de múltiples cables de red.

Patch cord

Cables cortos utilizados para conectar dispositivos dentro de una red.


Canaletas

Estructuras que organizan y protegen cables de red.

¿Que son
los componentes activos en la Red?

Son todos aquellos componentes que dependen de una fuente de energía. Por lo general pueden excitar un circuito inyectando energía en el circuito. Estos componentes pueden ser transistores, diodos, amplificadores. señales electrónicas y energía eléctrica.

Componentes Activos:

Switch

Dispositivo de red que conecta varios dispositivos dentro de una red local (LAN), facilitando la comunicación eficiente al dirigir datos entre ellos.

Router

Dispositivo que conecta diferentes redes y enruta el tráfico entre ellas, permitiendo la conexión a Internet y la comunicación entre redes locales y externas.

Tarjeta de red

Componente hardware que permite a un dispositivo conectarse a una red, ya sea a través de cable (Ethernet) o de forma inalámbrica (Wi-Fi).

Interruptor inteligente Wifi

Dispositivo que permite controlar aparatos eléctricos de forma remota a través de una red WiFi, integrándose en sistemas domóticos.

Access Point

Dispositivo que extiende la cobertura de una red inalámbrica, permitiendo a más dispositivos conectarse a la red dentro de un área determinada.

Herramientas:

Ponchadora

Herramienta para conectar cables en conectores RJ45 o similares.

Crimpadora

Similar a la ponchadora, asegura conectores en los extremos de cables.

Desforradora

Herramienta que quita el recubrimiento de cables sin dañar los conductores.

Tester probador de cables

Dispositivo que verifica la continuidad y el correcto funcionamiento de cables de red.
